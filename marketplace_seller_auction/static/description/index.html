Odoo Marketplace Seller Auction

Odoo Marketplace Seller Auction is a module which organizes online product auction on your Odoo Marketplace. The module facilitates your seller to host an auction for their marketplace products. The customers can place their bids on currently running auctions.

The details and status of every marketplace auction such as current bids, winners, etc. are available for the admin to view in the Odoo backend.

User Guide

Highlighted Features

  • This module provides a facility of bidding on any of their product of marketplace.
  • Buyers can bid for that product and winner can buy the product on that bid amount.
  • Admin can add auction on any product of marketplace.
  • Sellers can add auction only on their own product of marketplace.
  • Admin and seller can add incremental bidding rule on auction.
  • Admin and seller can set the status of an auction as running, completed or stopped.
  • A seller can manage all the settings for an auction of their own products.

Seller View of Odoo Marketplace Seller Auction
New Auction menu will be visible to a seller under the seller dashboard.

Sellers can view all the product auction records created by them.

Seller can enter the details of auction.

Auction on product has been started on website.

The seller can view all the completed/ closed/ soldout auction records.

A seller can view all the bidding increment rules.

Admin View of Odoo Marketplace Seller Auction
New Auction menu will be visible to admin under seller dashboard.

Admin can view all the product auction records created by sellers.

Admin can enter the details of the auction. Also, admin can confirm

this auction and can add the bid increment rule.

Auction on product has been started on website.

Admin can view all the completed/closed/soldout auction records

records of different sellers. On completion, if the auction product is

sold out, then the sale order will be linked in the auction record.

Here, Admin can create and view all the bidding increment rules for sellers.

Configuration View of Odoo Marketplace Seller Auction
Admin can manage the seller-wise allocation of the auction from the seller profile.

Admin can also allow auction feature for all sellers by

enabling default setting "Allow seller auction".
